

#### Vue2

```
实现原理
		对象类型：通过Object.defineProperty对属性的读取、修改进行拦截（数据劫持）
		数组类型：通过重写数组的一些列方法（如push）

存在问题（Vue3不存在以下两个问题）
		新增属性、删除属性，页面不会更新（可以通过Vue.set Vue.delete，或者this.$set this.$delete）
		直接通过下标修改数组，页面不会更新
```





#### Vue3

```
实现原理
		通过Proxy（代理），拦截对象中任意属性的变化，包括属性的增删改查
		通过Reflect（反射），对源对象的属性进行操作
```
